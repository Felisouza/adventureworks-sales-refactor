# Power BI - Adventure Works Sales Refactor

Este repositório tem como **único intuito o estudo** e prática de Business Intelligence com Power BI.  
O projeto consiste em **remodelar o painel Adventure Works**, disponibilizado pela Microsoft como dataset de exemplo, e **criar novos insights** a partir dos dados de vendas de bicicletas esportivas e acessórios.

---

## 📊 Sobre o Adventure Works
O Adventure Works é uma empresa fictícia de **venda de bicicletas esportivas e produtos relacionados**.  
O dataset inclui informações sobre:
- Vendas por produto, categoria e região
- Performance por canal de distribuição
- Lucro, margem e tendências de clientes

---

## 🎯 Objetivos do projeto
- **Estudo e prática** de modelagem de dados no Power BI.
- **Refatoração** do painel original para aplicar boas práticas de design e visualização.
- **Criação de novos insights** e KPIs relevantes para análise de vendas.
- Documentar o processo de transformação para fins de aprendizado.
